# wcat

## cat clone

```shell

npm link

```

### Commands

```shell
1. **wcat <filename>**
    ├───displays content of the file

2. **wcat <filenames>**
    ├───displays content of all the files


3. **wcat -s <filenames>**
    ├───convert big line breaks into a single line break

4. **wcat -n <filenames>**
    ├───give numbering to all lines

5. **wcat -b <filenames>**
    ├───give numbering to non-empty lines

Note: -n and -b can't be used together

```
